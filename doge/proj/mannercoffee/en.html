Manner Coffee

Website revamp project focused on HTML/CSS learning and optimization

#Website Revamp

Project Overview

This is a personal website revamp project based on Manner Coffee's official site, aimed at helping me practice and enhance my HTML/CSS skills. By optimizing layout and visual elements, I gained a better understanding of webpage structure, responsive design, and practical front-end techniques

The Problem

Although Manner Coffee operates over 1,800 stores, its website falls short in terms of information clarity and user experience, failing to reflect the scale and professionalism of the brand. Through this project, I wanted to analyze and redesign the site to create a cleaner, more visually appealing, and user-friendly experience that better represents the brand online

My Role

  • Optimized layout and implemented responsive design
  • Translated prototype into code using HTML/CSS/JS
  • Designed and replaced necessary visual elements
  • Completed the deployment setup for the website

Analysis

Finding①

Finding②

The key hero section features a blurry image that fails to convey the brand’s identity

Finding③

The product categories don’t highlight any unique selling points

Finding④

The store showcase section has potential, but it could be improved with store addresses and better photography for selected locations

Finding⑤

After the store section, there's an “About Us” segment that lists all store info in an infinite scroll. The excessive text doesn’t offer much value and results in a poor browsing experience
